## Category I: Philosophy & Ethics
**Purpose:** Philosophical reasoning, ethical analysis, conceptual exploration, argument deconstruction
**Skeleton:**
1. Philosophical Context
   - Identify relevant philosophical traditions
   - Define key concepts and terms
   - Establish historical context

2. Ethical Dimensions (if applicable)
   - Identify ethical stakeholders
   - Apply moral frameworks (utilitarianism, deontology, virtue ethics, care ethics)
   - Analyze moral conflicts

3. Argument Analysis
   - Reconstruct arguments (premises → conclusions)
   - Identify assumptions and implications
   - Evaluate logical coherence

4. Multiple Perspectives
   - Present different philosophical viewpoints
   - Compare and contrast approaches
   - Identify synthesis opportunities

5. Practical Application
   - Derive actionable insights
   - Connect to real-world implications
   - Suggest further exploration

6. Quality Validation
   - Check for conceptual clarity
   - Ensure fair representation of views
   - Balance depth with accessibility

## Usage Notes
- Always classify tasks into primary category first
- Use secondary categories when multiple dimensions exist
- Adapt skeletons based on specific task requirements
- Maintain consistent output structure within categories
